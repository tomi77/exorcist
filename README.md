# exorcist


## Założenia

* Platformówka / left-to-right
* rozwój od zakonnika do papieża
* znajdowanie starożytnych ksiąg z zaklęciami. każde zaklęcie to znak, który gracz musi wypowiedzieć
* rzucanie zaklęć, to wypowidanie ich
* później księgi z meta-zaklęciami (np. dwa znaki na raz). zasada ta sama, rzucanie, to wypowiadanie tych zaklęć
* dodatkowo księgi z informacjami o demonach (które zaklęcie / meta-zaklęcie działa najlepiej)
